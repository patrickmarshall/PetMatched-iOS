# PET MATCHED - iOS
A Tinder like app for your Pet to find a partner for your pet!

## Usage

1. Clone Project
2. Pod install

## Dependency

1. Alamofire
2. SwiftyJSON
3. HexColors
4. SwiftMessages
5. SDWebImage
6. SkyFloatingLabelTextField
7. PopupDialog
8. SBPickerSelector
9. Cloudinary
10. AlignedCollectionViewFlowLayout
11. Koloda
12. DPMeterView
13. IQKeyboardManagerSwift

## Minimum Requirements

1. iOS 10.0 or higher
2. Swift 4
3. XCode 9
